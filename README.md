## 📱 Overview

CurvyFitPT+ is a comprehensive fitness and wellness application created by **Maggie Bermudez**, a powerhouse personal trainer with over six years of experience and a professional athlete with the Women's National Football Conference (WNFC). This app helps users create and maintain a calorie deficit in a healthy, sustainable way through intelligent tracking, expert workout programs, and balanced meal plans.

## ✨ Features

### 🔥 Smart Calorie Deficit Tracker
- **Personalized BMR/TDEE calculation** based on age, weight, height, gender, and activity level
- **Meal tracking by category**: Breakfast, Lunch, Dinner, Snacks
- **Exercise logging** with calories burned
- **Real-time progress visualization** with animated progress ring
- **Calendar history** with color-coded indicators (green = deficit met, orange = close, red = over)
- **Date navigation** to view and edit any day's data
- **Local storage persistence** - your data is saved automatically

### 💪 Expert Workout Programs
Three professionally designed workouts with full details:

1. **Upper Body Ignite** (Beginner, 25 min, 200 cal, Home)
   - Circuit training with weighted and bodyweight options
   - Focus on form, posture, and steady pacing

2. **Leg Day Inferno** (Intermediate, 35 min, 300 cal, Gym)
   - Lower-body strength with machines and free weights
   - Targets quads, hamstrings, and glutes

3. **HIIT Speed Surge** (Advanced, 30 min, 350 cal, Home)
   - High-intensity interval training
   - Treadmill and non-treadmill options available

### 🥗 Balanced Meal Plans
13 delicious, macro-balanced meal options across 4 categories:

- **Power Breakfast** (3 options) - Protein-packed morning meals
- **Lean & Green Lunch** (3 options) - Balanced midday nutrition
- **Protein Power Dinner** (3 options) - High-protein evening meals
- **Smart Snacks** (4 options) - Quick, satisfying snacks

Each meal includes:
- Complete ingredient lists
- Calorie and macro breakdowns
- Preparation instructions
- Multiple alternatives to choose from

### 📱 Social Media Integration
- Direct links to Maggie's **YouTube**, **Instagram**, and **TikTok**
- Clickable social media icons with hover effects
- Access to daily workout demos, meal prep inspiration, and transformation stories

### 👤 About Section

### 🎨 User Experience

## 🛠️ Installation

### Quick Start (No Installation Required)

CurvyFitPT+ is a **single-file HTML application** with all assets embedded. Simply:

1. **Download** the `curvyfitpt-app.html` file
2. **Open** it in any modern web browser
3. **Start using** the app immediately!

### Clone the Repository

```bash
# Clone this repository
git clone https://github.com/yourusername/curvyfitpt-plus.git

# Navigate to the directory
cd curvyfitpt-plus

# Open the app
open curvyfitpt-app.html
# or
start curvyfitpt-app.html
# or simply double-click the HTML file
```

### Host on GitHub Pages

1. Fork this repository
2. Go to Settings → Pages
3. Select main branch as source
4. Your app will be live at `https://yourusername.github.io/curvyfitpt-plus/`

## 💻 Usage

### First Time Setup

1. **Sign Up** - Create an account with username, email, and password
2. **Profile Setup** - Enter your age, weight, height, gender, and activity level
3. **Get Your Goals** - The app calculates your personalized calorie target
4. **Start Tracking** - Log your first meal and see your progress!

### Daily Workflow

1. **Track Meals** - Add breakfast, lunch, dinner, and snacks throughout the day
2. **Log Workouts** - Record exercises with duration and calories burned
3. **Monitor Progress** - Check your calorie ring to see if you're on target
4. **Review History** - Use the calendar to view past days and track trends

### Features Guide

#### Calorie Tracking
- Click the date to access the **calendar view**
- Use **arrow buttons** to navigate between days
- Click **"+ Add"** on any meal category to log food
- Each entry shows item name and calories
- **"Clear All"** removes all entries for that category
- Delete individual items with the **× button**

#### Meal Plans
- Browse **4 categories** of meal plans
- Click any meal to see **3 detailed options**
- Each option includes ingredients, macros, and instructions
- Use ideas for meal prep or quick daily planning

#### Workouts
- View **3 workout programs** at different difficulty levels
- Click any workout for **full exercise breakdown**
- Each workout offers **weighted and bodyweight alternatives**
- Follow the structure for home or gym training

## 🎨 Customization

### Modify Colors

The app uses CSS variables for easy theming. Update these in the `<style>` section:

```css
:root {
    --primary-red: #C41E3A;      /* Main brand color */
    --dark-red: #8B0000;         /* Darker shade */
    --accent-red: #FF4757;       /* Bright accent */
    --black: #0A0A0A;            /* Background */
    --dark-gray: #1A1A1A;        /* Cards */
    --mid-gray: #2A2A2A;         /* Inputs */
    --light-gray: #3A3A3A;       /* Borders */
}
```

### Add Meal Plans

Extend the `mealPlansData` array in the JavaScript section:

```javascript
const mealPlansData = [
    {
        title: "Your New Meal Plan",
        intro: "Description here",
        options: [
            {
                name: "Meal Name",
                calories: "~400 cal",
                protein: "30g protein",
                fat: "15g fat",
                ingredients: ["ingredient 1", "ingredient 2"],
                instructions: "How to prepare..."
            }
        ]
    }
];
```

### Add Workouts

Extend the `workoutData` array:

```javascript
const workoutData = [
    {
        title: "Your Workout",
        level: "Intermediate",
        duration: "30 min",
        calories: "250 cal",
        location: "🏠 Home",
        description: "Workout description...",
        structure: {
            format: "Circuit (3 rounds):",
            exercises: [
                {
                    title: "Exercise Type",
                    options: ["Option 1", "Option 2"]
                }
            ],
            finisher: "Cool down exercise"
        }
    }
];
```
